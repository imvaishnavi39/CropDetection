## 🚀 NEXT STEPS - How to Run

### Step 1: Install Dependencies
Open PowerShell/CMD in the project directory:

```bash
# Activate virtual environment
venv\Scripts\Activate.ps1  # PowerShell
# OR
venv\Scripts\activate      # CMD

# Install all dependencies
pip install -r requirements.txt
```

### Step 2: Create the AI Model
```bash
python create_model.py
```

Expected output:
```
Creating placeholder disease detection model...
✓ Model saved successfully to: model/plant_model.pkl
```

### Step 3: Run Django Server
```bash
python manage.py runserver
```

Expected output:
```
Starting development server at http://127.0.0.1:8000/
Quit the server with CTRL-BREAK.
```

### Step 4: Open in Browser
Navigate to: **http://127.0.0.1:8000/**

## 🔧 Customization Guide

### Use Your Own Trained Model
1. Save your Keras model to `model/plant_model.h5`
2. Update `DISEASE_LABELS` in `Crop_Detection/model_loader.py`:
   ```python
   DISEASE_LABELS = ['Your', 'Disease', 'Classes', 'Here']
   ```
3. Ensure input size is 224x224 (or update preprocess_image function)
4. Restart the server

### Add More Disease Classes
Edit `Crop_Detection/model_loader.py`:
```python
DISEASE_LABELS = [
    'Healthy',
    'Powdery Mildew',
    'Rust',
    'Leaf Spot',
    'New Disease'  # Add here
]
```

### Modify UI Styling
All templates use Tailwind CSS. Edit:
- `Crop_Detection/templates/index.html` - Upload page styling
- `Crop_Detection/templates/result.html` - Results page styling
- `Crop_Detection/templates/base.html` - Global styling

### Change Input Image Size
Edit `Crop_Detection/model_loader.py`:
```python
def preprocess_image(image_path, target_size=(224, 224)):  # Change size here
```
