About this project

CropCare AI

A clean, college-ready web application that detects common leaf diseases from an uploaded image and returns a label with confidence and recommendations.

What is it?

CropCare AI helps users upload a plant leaf image and instantly see a prediction like Healthy, Powdery Mildew, Rust, or Leaf Spot — along with confidence and class-wise probabilities.

How it works
Step 1
Upload an image
Choose a JPG/PNG/GIF under 5MB.
Step 2
Preprocess
Resize to 224×224 and normalize RGB.
Step 3
Predict
Model returns probabilities for each class.
Step 4
Show results
Confidence, scores, and recommendations.

Technologies used

Simple stack, easy to explain in viva.
Django

Routing, templates, form validation, sessions.

Model loader

Loads a placeholder `.pkl` model (optional `.h5`).

Pillow + NumPy

Image resize/normalize and prediction formatting.

Tailwind (CDN)

Modern UI components without build tooling.

Presentation tip

Focus your explanation on the pipeline: upload → preprocess → predict → show confidence + recommendations.
