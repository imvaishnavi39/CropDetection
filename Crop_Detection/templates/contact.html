Quick FAQ
Short answers for demo/viva.
How accurate is CropCare AI?

For the placeholder model, results are for demonstration. With a trained dataset/model, accuracy improves significantly.

Is my data safe?

Uploaded images are stored under the local `media/` folder in development. You can clean them anytime.

Which crops does CropCare AI support?

The demo model supports 4 classes (Healthy, Powdery Mildew, Rust, Leaf Spot). You can expand by training a multi-class model.

Do you offer API access?

Not in this version. It can be extended with a REST API endpoint for mobile apps.

Is there a mobile app?

The UI is responsive, so it works well on phones. A dedicated app can be built later.
